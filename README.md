# Horses 4 Courses

I built a little horse racing game built with Angular as a fun way to refresh on signal-based state management with `@ngrx/signals`.

See the [Demo here](https://thedevelolper.github.io/horses-4-courses/)

## Roadmap

- [x] Prototyping the game
- [x] Creating a component library for the game (`ui-components`)
- [x] Deploying to GitHub Pages using GitHub Actions
- [ ] A Leaderboard component that reads from the state

## Getting Started

1. Download the repo
2. Perform an `npm install`
3. Run the app with `npm start`
4. Run storybook with `npm run storybook`
